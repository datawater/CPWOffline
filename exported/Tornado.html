Tornado,
an UCI compliant chess engine by Engin Üstün, its development started in 2004. Able to play Chess960, Tornado participated at the Chess960CWC 2006, and further at various CCT Tournaments, first was the CCT8 in 2006. Tornado NNUE, released in December 2020, features NNUE via Daniel Shawul's Scorpio NNUE egbbdll probing library [2] [3], using any Stockfish NNUE compatible weight file.

Features

[4]

  • Magic Bitboards
  • Multi-Threaded Search up to 64 cores
  • Iterative Deepening
  • Principal Variation Search
  • Aspiration Windows
  • Transposition Table
  • Selectivity
    • Null Move Pruning with Verification
    • Razoring
    • History Leaf Pruning
    • Futility Pruning
    • Late Move Reductions
    • Check Extensions
    • One Reply Extensions
    • Singular Extensions
    • Passed Pawn Extensions (either after establishing a new passer after a capture, or push to the 7th rank)
  • Move Ordering
    • Hash Move
    • Internal Iterative Deepening
    • Static Exchange Evaluation
    • MVV/LVA
    • Killer Heuristic
    • History Heuristic
  • Nalimov Tablebases
  • Evaluation
    • NNUE (Tornado NNUE)
