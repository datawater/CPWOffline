Nova Chess [1]

Nova Chess,
a MS-DOS shareware chess program by Thomas Starke, released in 1996. It comes with an own 2D graphics board graphical user interface. Nova Chess is hosted by the Internet Archive and can be executed via a browser-based DOSBox emulation [2].
