RumbleMinze,
an UCI compliant open source chess engine by Erik Madsen, written in C# to run under the Windows .NET framework. It was released in April 2012 [6], and is predecessor to MadChess which followed a few months later [7] introducing a new move generation technique using the C# yield statement [8] [9] [10] [11].
