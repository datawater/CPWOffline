Pirarucu [1]

Pirarucu,
an UCI compliant open source chess engine by Raoni Campos, written in Kotlin by JetBrains, which is designed to interoperate with Java. Therefor the engine runs on JRE (Java Runtime Environment) version 8 and above. Pirarucu's source code is licensed under the GPL v3.0.
