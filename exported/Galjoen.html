Galjoen [1]

Galjoen,
an open source chess program by Werner Taelemans, written in C++11 and distributed under the GNU General Public License, first released in February 2015 [2]. Galjoen comes with an own GUI build on gtkmm3, but can also ran as pure engine using UCI. Galjoen 0.35, released in March 2017 is able to play Chess960 [3].
