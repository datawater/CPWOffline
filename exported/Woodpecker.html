Ivory-billed Woodpecker [1]

Woodpecker,
a Chess Engine Communication Protocol compliant open source chess program by Evgeniy Korniloff, written in C, first released in 2007 as a branch of OBender 3.1.0. with almost the same board representation but using some bitboards for pawn and evaluation stuff, for instance concerning (open) diagonals and files. Woodpecker further uses centipawn resolution instead of OBender's millipawns [2].
