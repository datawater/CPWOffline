TCEC Season 17,
the Top Chess Engine Championship (TCEC) Season 17 from January 02 until April 21, 2020. It was the first time, TCEC differentiated between a CPU League and a GPU League - the latter cancelled so far, since only two participants met the uniqueness criteria, to become default qualifiers for the Playoff for the Premier Division along with the four top engines of CPU League 1. The tournament hardware was enhanced for the CPU server with 4 x Intel Xeon 4xE5-4669v4 @ 2.8 GHz with 88 physical cores, and the GPU Server with 2x Intel Xeon E5-2630V4 @ 2.2 GHz and 4 x Nvidia RTX 2080 ti. Engines were running under Linux, either natively or as Windows engine under the compatibility layer Wine. 6-man Syzygy bases were cached in RAM [2].

Superfinal : LCZero v0.24-sv-t60-3010 - Stockfish 20200407DC 52½-47½
