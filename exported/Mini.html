Mini,
a chess program by Don Dailey and Larry Kaufman which played the WCCC 1999 in Paderborn, operated by Harald Prokop. It was used to even the number of entries, based on the 1993 C program with the internal name Mini, the common ancestor of the ACM 1993 winner Titan aka Socrates II (C), the commercial Socrates 3.0 (Assembly), and Kasparov's Gambit (C) [2].

WCCC 1999

The WCCC entry was a quick modification in order to get it working on Unix without GUI. Apparently, there were time control issues since Mini played almost instantly, additionally during the first rounds the hash table was set too high, so the disk was trashing.
