Clubfoot,
a very basic UCI open source chess engine by Shawn Chidester, written in C++. It was started as a real-world example of how to implement a UCI chess engine using the Senjo UCI adapter [2]. It uses 0x88 board representation, null move pruning, PVS with aspiration windows, iterative deepening, IID, transposition table, killer heuristic, check extensions, and late move reductions using a custom version of the history heuristic - among other common concepts.

This engine is available as source code [3], and Dann Corbit has provided binaries [4]. Links to 32 and 64 bit binaries for Windows and Linux are also available in the Clubfoot README, found in the github source repository [5].
