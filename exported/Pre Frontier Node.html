Pre Frontier Node

Home * Search * Node * Pre Frontier Node

Pre Frontier nodes are nodes immediately above Frontier Nodes, typically being searched at depth of 2. This is where extended futility pruning might be applied. The definition is taken from the papers of Ernst A. Heinz [1]. Robert Hyatt defines pre frontier nodes as depth == 1 nodes [2].

References

  1. ↑ Ernst A. Heinz (1998). Extended futility pruning. ICCA Journal, Vol. 21, No 2, ps
  2. ↑ Re: simple node definitions question by Robert Hyatt, CCC, September 13, 2004
