Phodopus Sungorus [1]

Sungorus,
an UCI compliant open source chess engine by Pablo Vazquez, first released in June 2009 [2]. Last recent version is Sungorus 1.4 from June 2010, which was base of further development by Pawel Koziol and his Rodent engine [3].

Features

Board Representation

  • 6+2 Bitboard Board Definition
  • 8x8 Board
  • Kindergarten Bitboards
  • BitScan by De Bruijn Multiplication
  • SWAR-Popcount

Search

  • Fail Soft Alpha-Beta with Principal Variation Search
  • Quiescence Search at depth <= 0
  • Transposition Table
  • Null Move Pruning
  • PV-Table on Stack
  • MVV/LVA for sorting captures
  • History Heuristic

Evaluation

  • Material
  • Piece-Square Tables
  • Mobility
