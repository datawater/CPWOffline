Jon Dart [1]

Jon Dart,
an American software developer, software architect, and independent IT consultant [2]. As a chess programmer he is author of the Chess Engine Communication Protocol and UCI compatible chess engine Arasan, available to run under Windows, Linux and Mac OS [3].
