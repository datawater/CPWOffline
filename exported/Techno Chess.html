Techno Chess, (TechnoChess)
a chess program by Christophe Jolly written in C to run under Windows. Its development started in 1996 [2], supposable incorporating all the good stuff from Christophe's earlier program BB, and soon evolved to KnightX, first published in 1999 supporting the Chess Engine Communication Protocol with Windows - and since 2004, Linux executables [3]. Techno Chess played the the WMCCC 1997 in Paris.
