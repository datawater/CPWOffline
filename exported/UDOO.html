UDOO Quad [1]

UDOO,
an open source hardware single-board computer with an ARM i.MX6 Freescale Cortex-A9 Dual/Quad core processor at 1GHz [2] plus Atmel SAM3X8E ARM Cortex-M3 CPU, Vivante GPU, 1 Gibibyte DDR3 RAM, 76 GPIO, HDMI, LVDS, Ethernet RJ45, Wi-Fi, Mini USB, SATA (quad core only), Micro SD and Arduino compatible interface. UDOO was first launched on Kickstarter on April 9, 2013 [3], and is able to run either Android or Linux [4].

Computer Chess

The UDOO is suitable to run Linux or Android compliant chess engines and to control Sensory boards to build own Dedicated Chess Computers. Folkert van Heusden ran PuppetMaster aka DeepBrutePos in "puppet master" mode on the UDOO, requesting best moves from external UCI or XBoard engines, each running on one of seven Raspberry Pi nodes [5].
