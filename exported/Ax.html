Đông Sơn Axe [1]

Ax, (AxChess) [2]
a WinBoard compatible chess engine by Pham Hong Nguyen, written in C++, first released May 24, 2002 [3] . The first version 0.1 used to allocate the transposition table around the half of physical memory size, if that fails one quarter. Subsequent versions feature configurable hash sizes [4] .
