Marco Costalba,
an Italian software developer and proponent of open source software [1]. Marco is author of QGit [2] , a GUI of the free distributed version control system Git [3], initially designed and developed by Linus Torvalds for Linux kernel development.

Stockfish

Based on Tord Romstad's open source chess program Glaurung under the GNU General Public License, he started a branch called Stockfish in 2008 [4], which in the meantime has become the main branch of Glaurung, further supported by Tord Romstad and Joona Kiiski as development team [5].
