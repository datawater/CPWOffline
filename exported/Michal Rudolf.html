Michał Rudolf [1]

Michał Rudolf,
a Polish FIDE master in chess [2], computer scientist and independent software developer with focus on GUI programming with Qt, Linux, Windows, C++, PHP, web development, open source software, linguistics, natural language processing and computer chess [3]. He contributed to SCID, and is active developer of the free multi-platform ChessX database and GUI. Along with Mark Uniacke, Michał Rudolf is co-author of the commercial HIARCS Chess Explorer [4].
