Wb2UCI, (Wb2Uci, WB2UCI)
a protocol adapter by Odd Gunnar Malin to run WinBoard Engines using the Chess Engine Communication Protocol under chess GUIs supporting UCI, such as the Shredder GUI. Wb2UCI is a console application running under Windows. Along with the executable Wb2Uci.exe, one must place a Wb2Uci.eng textfile into the engine folder [1] [2].

Wb2Uci.eng

The Wb2Uci.eng file consists of following mandatory attribute–value pairs [3], and many more optional. 

;---------------------------
[ENGINE]
Name=Name of engine
Author=Name of engine author
Filename=Wb2Uci.exe

[OPTIONS]
Program=engine.exe
;---------------------------
