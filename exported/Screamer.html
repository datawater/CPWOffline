Screamer, (Screamer Chess)
a chess program for the Macintosh by William Bryant, written in C++ using the CodeWarrior IDE, supporting PowerPC and early versions prior to build 57, 680x0 processors. Screamer comes with an own GUI based on the PowerPlant toolkit, an own opening book, and supports the 3-man, 4-man, and 5-man Nalimov tablebases, and a variable transposition table size. The program applies negamax alpha-beta improved by PVS, aspiration windows inside an iterative deepening framework, and recursive null move pruning [2].
