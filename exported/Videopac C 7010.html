Videopac G7000 with C 7010 chess module [1]

Videopac C 7010,
a chess module for the Philips Videopac G7000 second generation Video game console, released in 1982. The C 7010 module had its own NSC800 CMOS microprocessor [2] , which is Z80-compatible [3] . The program is based on Gambiet 80 by Wim Rens and Microtrend [4] .
