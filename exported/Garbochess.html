Garbo in Inspiration [1]

Garbochess,
an UCI compliant open source chess engine by Gary Linscott. Starting with a 0x88 board representation in C#, which later evolved to the JavaScript version Garbochess-JS [2] aka Ajax chess [3], and was also incorporated in Silverlight Chess [4] for the Microsoft Silverlight framework, Version 2 was a complete rewrite in C++ using bitboards [5].
