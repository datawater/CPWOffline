Numpty chess,
a free Chess Engine Communication Protocol (aka WinBoard) compatible chess engine by Chris Tatham. Having initially been started in JustBasic [1] it was subsequently translated into, and developed in FreeBASIC. Initially released in December 2005, Numpty has always had an open source code base. It is a comparatively weak engine, having a primary focus of being a reasonable but beatable opponent to casual and club players [2] .

Features

[3]

  • 10x12 array based board structure with pseudo legal move generation
  • Alpha-beta search with iterative deepening and limited quiesence
  • Simple evaluation including material, game phase piece-square tables and basic endgame knowledge
