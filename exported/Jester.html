Jester,
a chess engine by Stéphane Nguyen, written in Java, distributed as helpware [2]. As Java applet, Jesper is launched from a web page and executed within a Java Virtual Machine (JVM) in a process separate from the web browser itself. Jester uses the ICS protocol [3] allowing it playing on free chess servers [4]. Jester played various French Computer Chess Championships and French Programmers Tournaments from 1998 until 2003.
