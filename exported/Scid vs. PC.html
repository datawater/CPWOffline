Scid vs. PC,
an open source Chess GUI with the ability to run detailed chess analysis, to create and query huge databases, to play on-line with FICS and against computer opponents, and to manage computer tournaments. It is a fork of Shane Hudson's Chess Information Database (SCID), started in 2009 by Steven Atkinson, and requires installation of Tcl/Tk [2]. Scid vs. PC runs under Linux, Mac OS and Windows computers, supporting Chess Engine Communication Protocol and UCI compliant engines.
