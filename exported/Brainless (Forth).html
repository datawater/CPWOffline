Brainless,
an open source chess program by David Kühling, released in 2010 under the GPL Version 3 [1]. Brainless is written in ANS Forth [2], and uses the Forth terminal via ANSI color terminals and Unicode chess piece glyphs for quasi-graphical display to interact with the user. It features a 10x12 Board, alpha-beta with transposition table, null move pruning, futility pruning and killer heuristic.
