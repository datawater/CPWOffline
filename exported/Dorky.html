Dork [1]

Dorky, (Dorky Chess)
a WinBoard compatible engine for Windows written by Matt McKnight in C/C++, first released in May 2003 [2]. Dorky already played the CCT2 in 2000, and Dorky 4.0 the CCT6 tournament. Dorky 4.x, a rewrite of most major components, was the first release of a new version after almost seven years in May 2011, with access to Nalimov Tablebases and a parallel search [3].

Features

Following algorithms and features are used in Dorky Chess 4.3 [4]

  • PVS alpha-beta with quiescence search
  • SMP search using Dr. Hyatt’s DTS algorithm
  • Nalimov Endgame tablebases
  • Late move reductions
  • Futility pruning
  • Transposition hash table / Pawn evaluation hash table (using Hyatt’s lockless approach)
  • Null-move pruning
  • Killer move heuristic
  • Bitboards, rotated bitboards for move generation
  • Static exchange evaluation
  • Opening Book creation and simple learning
